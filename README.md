# Caiden Durham Portfolio (Lab 2)

This repository contains my Lab 2 portfolio website project for Southern Illinois University.

## Project Files

- `index.htm` — Portfolio home page (updated with external CSS).  
- `interests.htm` — Interests page using a two-column layout.  
- `styles.css` — Shared stylesheet for both pages.  
- `images/` — Image assets used by the website.

## Lab 2 Requirements Implemented

### Part 1 (Index Page + CSS)
- External stylesheet linked from `index.htm`.
- 10px page margin.
- 10px padding on headings.
- 5px left/right padding on paragraphs.
- Body width set to 725px.
- 10px left padding on list items.
- Blue border with rounded corners on profile image only.
- Image positioned using CSS `position`.
- Light background color applied.
- CSS file includes a comment header and comments.

### Part 2 (Interests Page)
- Created `interests.htm`.
- Implemented a professional two-column layout.
- Each column includes two sections (4 total sections).
- Reused `styles.css` from Part 1.
- Includes comment header and comments in HTML.
